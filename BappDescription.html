Wordlist Importer is a Burp extension, written in Python, that enables a smooth import of wordlists hosted on the Internet right into Burp Suite.

Features

  • Import a wordlist from a URL or a local file.
  • Display a sample of 100 words after importing the wordlist.
  • Add the URLs and file paths to the Wordlist History.
  • Use the imported wordlist in Intruder by selecting the Payload type as Extension-generated.
  • Merge wordlists from multiple URLs or file paths, with automatic deduplication.
  • Export the merged wordlists onto a local storage.
  • Clear the loaded wordlist in memory using the Clear Wordlist option.
  • Clear the Wordlist History using the Clear History option.
  • Delete a specific wordlist from the history using the Delete option.

Usage

Importing a Wordlist from a URL

  1. Enter the URL of a raw text file containing the wordlist.
  2. Click on Import. The wordlist gets loaded, and a preview of 100 words is displayed.
  3. The URL is then added to the Wordlist History.

Importing a Wordlist from a Local File

  1. Enter the path of a local file, or hit the Choose File option to browse and select one.
  2. Click on Import. The wordlist gets loaded, and a preview of 100 words is displayed.
  3. The file path is then added to the Wordlist History.

Using the Imported Wordlist in Intruder

  1. Select the Payload type as Extension-generated in the Intruder tool.
  2. Use the Select generator option to select the Wordlist Importer extension.
  3. Hit the Start Attack option to use the wordlist imported from the extension.

Merging Wordlists

  1. Select two or more URLs and/or file paths by checking their respective checkboxes.
  2. Hit the Merge Wordlists option to integrate the wordlists and remove duplicate entries.
  3. Export the merged wordlists if you want to save a local copy.

Additional Operational Details

  • Clear the wordlist stored in memory using the Clear Wordlist option.
  • Clear the Wordlist History using the Clear History option.
  • Delete a particular wordlist from the history using the Delete option.
