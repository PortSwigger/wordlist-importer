Wordlist Importer streamlines the process of importing wordlists into Intruder attacks by allowing direct access to both remote URLs and local files from a dedicated extension tab. The extension provides comprehensive wordlist management capabilities including merging, deduplication, and history tracking for efficient payload generation workflows.

Features

  • Direct import of wordlists from remote URLs or local file paths
  • Live preview of imported wordlists with sample display of first 100 entries
  • Seamless integration with Intruder as an extension-generated payload source
  • Advanced wordlist merging with automatic deduplication across multiple sources
  • Persistent wordlist history with checkbox selection for batch operations
  • Export functionality for saving merged wordlists to local storage
  • Memory management with clear wordlist and history cleanup options
  • File browser integration for convenient local wordlist selection

Usage

  1. Navigate to the Wordlist Importer tab
  2. Enter a URL pointing to a raw wordlist or use Choose File to select a local file
  3. Click Import to load the wordlist and display a 100-word preview
  4. In Intruder, set payload type to Extension-generated and select Wordlist Importer as the generator
  5. For multiple wordlists, check desired entries in the history and use Merge Wordlists to combine with deduplication
  6. Export merged wordlists using the export function to save locally for future use
